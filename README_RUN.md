# Lancement du projet

1. Supprimez tout ancien `node_modules` si présent.
2. Exécutez `npm install`.
3. Configurez `.env` à partir de `.env.example`.
4. Lancez la migration: `npm run migrate`.
5. Démarrez: `npm run dev` ou `npm start`.


## Variables APIs paiement
WAVE_API_URL=https://api.wave.com
WAVE_API_KEY=your_key
OM_API_URL=https://api.orange-sonatel.com
OM_API_TOKEN=your_token
